## 前言

这个博客之前是用 Hugo 搭建的，但是很久没有维护了。最近决定重新整理一下，把博客重新上线。

## 技术栈

这次博客使用的技术栈：

| 组件 | 技术 |
|------|------|
| 静态生成器 | [Hugo](https://gohugo.io/) v0.120 |
| 主题 | [PaperMod](https://github.com/adityatelange/hugo-PaperMod) |
| 托管 | [GitHub Pages](https://pages.github.com/) |
| 域名 | bigjeager.github.io |

## 为什么选择 Hugo？

1. **速度快** - 构建速度非常快，几百篇文章也是秒级生成
2. **简单** - Markdown 写作，无需数据库
3. **免费** - GitHub Pages 免费托管
4. **主题丰富** - 有很多漂亮的主题可选
